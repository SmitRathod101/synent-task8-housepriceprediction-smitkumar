# House Price Prediction using Machine Learning

## Project Overview

This project develops a Machine Learning model to predict house prices using the Ames Housing Dataset. Linear Regression is used to learn the relationship between important housing features and property sale prices.

## Objective

* Perform data preprocessing
* Handle missing values
* Select important features
* Train a Linear Regression model
* Evaluate model performance
* Generate house price predictions

## Dataset

Dataset: Ames Housing Dataset (House Prices)

Target Variable:

* SalePrice

Features Used:

* OverallQual
* GrLivArea
* GarageCars
* TotalBsmtSF
* FullBath
* YearBuilt

## Workflow

1. Data Loading
2. Missing Value Analysis
3. Data Cleaning
4. Feature Selection
5. Train-Test Split
6. Model Training
7. Model Evaluation
8. Visualization
9. Conclusion

## Model Used

* Linear Regression

## Results

Model Performance:

* RMSE: 39,710.99
* R² Score: 0.7944

The model explains approximately 79% of the variation in house prices, demonstrating strong predictive capability.
